# 集中营打卡

一个多人共享的健身打卡小站。用户输入姓名与训练心得后，服务端自动记录打卡时间，所有访问同一部署地址的人都能看到记录。

## 本地运行

需要 Node.js 18 或更高版本：

```bash
npm start
```

然后打开 <http://localhost:3000>。打卡数据默认保存在 `data/checkins.json`。

## 部署

可部署到任何支持常驻 Node.js 服务的平台。启动命令为 `npm start`，平台通过 `PORT` 环境变量指定端口。生产环境请为 `data` 目录配置持久化磁盘，或通过 `DATA_DIR` 指向持久化目录。
